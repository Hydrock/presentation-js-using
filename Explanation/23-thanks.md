## Как провести Q&A

### Типичные вопросы

**Q: Можно ли использовать using уже сейчас?**
A: Да, с TypeScript 5.2+ или в современных браузерах и Node.js 20.10+.

**Q: Что делать, если среда не поддерживает using?**
A: Используйте транспиляцию в try-finally через TypeScript или Babel.

**Q: Влияет ли using на производительность?**
A: Нет, это синтаксический сахар. Транспилированный код идентичен ручному try-finally.

**Q: Когда стоит использовать using, а когда try-finally?**
A: Используйте using для нового кода в поддерживаемых средах. try-finally — для совместимости.

**Q: Как добавить поддержку using в свой класс?**
A: Реализуйте метод `[Symbol.dispose]()` для синхронного или `[Symbol.asyncDispose]()` для асинхронного освобождения.
