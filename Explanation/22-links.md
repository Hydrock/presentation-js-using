# Слайд 22: Полезные ссылки

## Описание слайда

Список ресурсов для углублённого изучения темы.

## Официальные ресурсы

### TC39 Proposal
https://github.com/tc39/proposal-explicit-resource-management

Официальный репозиторий предложения с:
- Полным текстом спецификации
- Историей обсуждений
- Примерами использования
- Статусом реализации

### TypeScript 5.2 Release Notes
https://www.typescriptlang.org/docs/handbook/release-notes/typescript-5-2.html

Официальная документация TypeScript о поддержке `using`:
- Описание возможностей
- Примеры кода
- Настройка компилятора

### MDN Documentation
https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Statements/using

Документация Mozilla Developer Network:
- Справочник по синтаксису
- Примеры использования
- Совместимость браузеров

### Can I Use
https://caniuse.com/?search=using

Актуальная информация о поддержке:
- Таблица браузеров
- Версии с поддержкой
- Глобальная статистика
