# Слайд 5: Аналоги в других языках

## Описание слайда

Сравнение механизмов управления ресурсами в различных языках программирования.

## Обзор аналогов

### C# — using (2003)
C# был первым mainstream-языком, который ввёл конструкцию `using` для детерминированного освобождения ресурсов.

```csharp
using (FileStream fs = File.Open("file.txt")) {
    // работа с файлом
} // автоматически вызывается fs.Dispose()
```

Любой объект, реализующий интерфейс `IDisposable`, может использоваться с `using`.

### Python — with (2006)
Python использует контекстные менеджеры с конструкцией `with`.

```python
with open('file.txt', 'r') as f:
    content = f.read()
# автоматически вызывается f.__exit__()
```

Контекстные менеджеры реализуются через методы `__enter__` и `__exit__`.

### Java — try-with-resources (2011)
Java 7 представила конструкцию try-with-resources.

```java
try (FileInputStream fis = new FileInputStream("file.txt")) {
    // работа с файлом
} // автоматически вызывается fis.close()
```

Объекты должны реализовывать интерфейс `AutoCloseable`.

### Rust — RAII / drop (2015)
Rust использует паттерн RAII (Resource Acquisition Is Initialization). Освобождение происходит автоматически при выходе из scope через систему владения.

```rust
{
    let file = File::open("file.txt")?;
    // работа с файлом
} // автоматически вызывается drop(file)
```

Особенность Rust — это происходит на уровне системы типов, без специального синтаксиса.

### Go — defer (2009)
Go использует ключевое слово `defer` для отложенного выполнения.

```go
file, _ := os.Open("file.txt")
defer file.Close()
// работа с файлом
// Close() вызовется при выходе из функции
```

## Сравнительная таблица

| Язык | Механизм | Интерфейс | Особенность |
|------|----------|-----------|-------------|
| C# | using | IDisposable | Явный синтаксис |
| Python | with | __enter__/__exit__ | Контекстные менеджеры |
| Java | try-with-resources | AutoCloseable | Интеграция в try |
| Rust | RAII | Drop trait | Неявный, на уровне типов |
| Go | defer | — | Отложенное выполнение |
