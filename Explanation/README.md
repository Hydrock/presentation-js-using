## Ключевые тезисы

1. `using` стандартизирован в ES2024
2. Решает проблему детерминированного освобождения ресурсов
3. Аналог using в C#, with в Python, try-with-resources в Java
4. Поддержка в TypeScript 5.2+, современных браузерах, Node.js 20.10+
5. Упрощает код и предотвращает утечки ресурсов
